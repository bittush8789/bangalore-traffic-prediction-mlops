ℹ️ About This Project

An AI-powered Smart Mobility Solution for Bengaluru's traffic challenges.

🎯 Project Overview

The Bangalore Traffic Prediction & Smart Route Intelligence System is an end-to-end machine learning web application that predicts traffic congestion, estimates travel times, recommends optimal departure times, and suggests the best routes across 20 key Bangalore locations.

📊 Dataset

We generated a synthetic dataset of 300,000+ records spanning 3 years (2023–2025) with realistic Bangalore traffic patterns including:

  • Morning & evening peak hour patterns
  • IT corridor (Whitefield, Electronic City, Manyata) rush
  • Weather impact (monsoon, fog, thunderstorm)
  • Event-driven traffic (IPL, concerts, festivals)
  • Accident and construction delays
  • Weekend vs weekday differences

🧠 Machine Learning Models

Classification (Congestion Level):

  • Random Forest, XGBoost, Gradient Boosting, Logistic Regression

Regression (Travel Time):

  • Random Forest, XGBoost, Gradient Boosting, Linear Regression

Forecasting (Traffic Index):

  • XGBoost Time-Series with lag features
